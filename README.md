# LuaPlayerYT-MCSM

## Как собрать?
1. Установите PSPSDK как [указано в инструкции](https://pspdev.github.io/installation.html).
2. Склонируйте в целевую папку репозиторий при помощи команды:
```bash
git clone https://github.com/entitybtw/LuaPlayerYT-MCSM
```
3. Откройте в терминале папку репозитория и запустите сборку:
```bash
./build.sh
```

*Примечание: в репозитории работает автосборщик EBOOT.PBP на каждое изменение. Версию, собранную из актуального кода, можно загрузить в [Actions](https://github.com/entitybtw/LuaPlayerYT-MCSM/actions?query=branch%3Amain), выбрав изменение.*
